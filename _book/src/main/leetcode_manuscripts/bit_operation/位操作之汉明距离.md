## 位操作之汉明距离

### 方法1

```java
public int hammingDistance(int x, int y) {
    int z = x ^ y;
    int res = 0;
    while (z != 0) {
        if ((z & 1) == 1) res++;//计算每个低位的1的个数
        z >>= 1;//抹掉1位
    }
    return res;
}
```

### 方法2

```java
        public int hammingDistance(int x, int y) {
            int res = 0;
            while (x != 0 || y != 0) {
                int bitx = x & 1, bity = y & 1;//去到x y的低位的结果，判断是否相等
                if (bitx != bity) res++;
                x >>= 1;
                y >>= 1;
            }
            return res;
        }
```

```java
        public int hammingDistance(int x, int y) {
            int z = x ^ y;
            int res = 0;
            while (z != 0) {
                res++;
                z -= z & (-z);
            }
            return res;
        }
```
