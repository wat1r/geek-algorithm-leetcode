## 双向BFS之单词搜索[Shih Tzu]

### 0.背景

> 双向bfs适用于知道起点和终点的状态下使用，从起点和终点两个方向开始进行搜索，可以非常大的提高单个bfs的搜索效率

#### 伪代码

```c++
while(!queue.empty()) {
    t=queue.front();
    queue.pop();
    for(每个t的儿子s) {
        if(正向搜索)
            if(vis1[s]==0) {//找到答案
                if(vis2[s]==1)
                    over;
                vis1[s]=1;
                queue.push(s);
            }
        else//反向搜索
            if(vis2[s]==0) {//找到答案
                if(vis1[s]==1)
                    over;
                vis2[s]=1;
                queue.push(s);
            }
    }
}
```

#### 逐层扩展

```c++
void bfs()
{
    for(int i = 1; i <= max; i++)
```
