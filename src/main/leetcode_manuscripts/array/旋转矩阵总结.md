arr[i][j] = nums[n-1-j][i]

arr[i][j] = nums[j][n-1-i]

#### [面试题 01.07. 旋转矩阵](https://leetcode-cn.com/problems/rotate-matrix-lcci/)

```java
        public void rotate(int[][] mat) {
            int N = mat.length;
            int[][] grid = new int[N][N];
            for (int i = 0; i < N; i++) {
                for (int j = 0; j < N; j++) {
                    //旋转90°，利用空间数组
                    grid[i][j] = mat[N - 1 - j][i];
                }
            }
            for (int i = 0; i < N; i++) {
                for (int j = 0; j < N; j++) {
                    mat[i][j] = grid[i][j];
                }
            }
        }
```
