## 我真的不会旋转矩阵

```java
//顺时针旋转90°
arr[i][j] = nums[n-1-j][i]
//顺时针旋转180°
arr[i][j] = nums[n-1-i][n-1-j]
//顺时针旋转270°
arr[i][j] = nums[j][n-1-i]
```
